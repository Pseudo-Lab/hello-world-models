Critic Loss — 수식 + 안정화 기법

Vλ를 target으로 squared error 회귀
단, target이 자기 자신을 쫓지 않도록 2가지 안정화 기법 필요
🎯 target Vλt — λ-return
Vλt = r̂t + γ̂t · [ (1−λ)·vξ(ẑt+1) + λ·Vλt+1 ]
(t < H)   ·   VλH = vξ(ẑH)
📉 Critic Loss
L(ξ) = 𝔼 [ H−1Σt=0 ½ (vξ(ẑt) − sg(Vλt))² ]
🛑sg(·) — stop-gradient
역전파 차단 — target 쪽으로 gradient 안 흐름.
Vλ 계산에 쓰인 값들은 상수로 취급.
🧊Target network — 100 step마다 복사
본체는 매 step 학습,
target용 복사본은 느리게 갱신 → 자기 자신을 쫓는 불안정성 제거 (Mnih 2015).
vξ(ẑt)가 sg(Vλt)에 가까워지도록 학습
마지막 step (t = H)은 loss 항 없음 (target = critic 자체)
