Multi-modal 분포란?

모드(mode) = 확률 분포의 봉우리 — 여러 봉우리면 multi-modal
1개
Uni-modal
봉우리 1개 Gaussian (DreamerV1)
여러 개
Multi-modal
봉우리 여러 개 Categorical (DreamerV2)
게임 상태가 여러 가능성을 동시에 가질 때가 multi-modal. 예: 방 A로 이동 vs 방 B로 이동 — 둘 다 가능성 있음 → Gaussian은 봉우리 1개뿐이라 둘 중간의 엉뚱한 상태를 예측하지만, Categorical은 여러 선택지를 그대로 표현할 수 있음.
