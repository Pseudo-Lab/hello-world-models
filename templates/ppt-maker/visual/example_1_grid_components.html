World Model 구성 요소 (DreamerV2)

입력 이미지: 84×84 grayscale → 64×64 downscale
Stochastic state: 32×32 categorical
Model state = deterministic h ⊕ sample z
모든 네트워크는 파라미터 φ로 통합되어 함께 학습
RSSM (Recurrent State-Space Model)
🔁
Recurrent Model
RSSM의 결정적 상태
ht = fφ(ht−1, zt−1, at−1)
GRU
이전 state와 action을 받아 결정적 순환 상태 ht 계산. 시간 방향 기억 담당.
👁️
Representation Model
Posterior — 이미지 반영
zt ∼ qφ(zt | ht, xt)
CNN + MLP
이미지 xt와 ht로부터 stochastic state zt를 추론. World Model 학습 시 사용.
🔮
Transition Predictor
Prior — 이미지 없이 예측
ẑt ∼ pφ(ẑt | ht)
MLP
ht만 보고 다음 z를 상상으로 예측. 상상 rollout의 핵심 — 이미지 없이 미래 생성.
Predictors (관측·보상·종료 복원)
🖼️
Image Predictor
Decoder — 이미지 복원
x̂t ∼ pφ(x̂t | ht, zt)
Transposed CNN
latent (h, z)로부터 원본 이미지 복원. reconstruction loss로 representation 학습 유도.
🏆
Reward Predictor
보상 추정 head
r̂t ∼ pφ(r̂t | ht, zt)
MLP
현재 model state에서 기대 보상 예측. 상상 rollout에서 reward signal 공급.
⏳
Discount Predictor
에피소드 종료 추정
γ̂t ∼ pφ(γ̂t | ht, zt)
MLP
에피소드가 끝날 확률을 예측. 상상 trajectory에서 bootstrap 경계 처리.
